# BACKLOG — 보류된 작업

> **무엇을 안 했는지** + **왜 안 했는지** 기록. 다음 AI(또는 미래의 사용자)가 같은 고민 반복하지 않게.

형식:
```
## [우선순위] 제목
- 무엇: ...
- 왜 보류: ...
- 재개 조건 / 참고: ...
```

우선순위: `P0`(긴급) · `P1`(곧) · `P2`(언젠가) · `P-`(영구 제외)

## P- 외부 번역 API 연동 (DeepL, Google Translate, OpenAI) — **재보류**

- **무엇:** 한글 요청을 자동으로 영어로 번역해 프롬프트에 삽입.
- **이력 (26-04-28):**
  1. v0.2까지 "최신 모델은 한국어 직접 이해" 가정으로 미사용.
  2. MJ/Niji 한국어 인식 약함 확인 → OpenAI API 도입을 v0.3 1차 작업으로
     일시 채택.
  3. ChatGPT 구독과 OpenAI API 결제가 별개라는 비용 부담 인지 + 무료 API
     대안의 한계 확인 → **자동 번역 자체를 v0.3에서 제거**, 사용자가
     ChatGPT 등으로 직접 번역해 영어 textarea에 붙여넣는 워크플로 채택.
- **현 상태:** 자동 번역 미사용. 영어 textarea로 사용자 직접 입력.
- **재개 조건:**

  - 무료/저렴한 번역 API의 게임 도메인 용어 처리가 검증될 때.
- **참고:** `docs/design-docs/PROMPT_STRATEGY.md` §2 (재개정판).
